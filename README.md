# HireMyTech
HireMyTech Assessment

## Run server
To run server in production
  1) cd or go to 'server' directory 
  2) run 'npm run build' in terminal
  3) run 'npm start'
  
 ## Run client
 To run client
  1) cd or got to 'client' directory
  2) run 'npm start'
  3) go to browser and type 'localhost:3000' in url search bar
